# Varonis
## Installation
### Installation Requirements
Computer must have VaronisManagement as an installable Module
### Installation steps
1. Copy Varonis-Powershell zip onto computer
1. Extract contents
1. Open Powershell Prompt
1. Change the Powershell Directory to the extracted folder location: `cd .\Varonis-Powershell\VrnsSupport`
1. Run the following command `Import-Module .\VrnsSupport.psm1`
